The model in this file cover the following region:

Extended Region:
Latitude:   9 N - 89.5 N           ( original: 20 N - 89.5 N )
Longitude: 56 W - 70 E             (          40 W - 70 E    )

Resolution is 0.5 x 0.5 degrees

EPcrust is in 3 layer:
- sediment
- upper crust
- lower crust

For completeness we also provide the ice layer thickness in the file Ice_thickness_0_5.txt

Thickness, P-speed, S-speed and density  of each layer, and topography are specified.

in particular the columns in the file are:

LON   LAT   Topography   sediment_thickness   thickness_upper_crust   thickness_lower_crust   VP_sediment   VP_upper_cruet   VP_lower_crust   VS_sediment   VS_upper_crust   VS_lower_crust   RHO_sediment   RHO_upper_crust   RHO_lower_crust


Please, refer to the paper for more informations.
